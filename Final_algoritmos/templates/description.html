Descripción del Proyecto en la App Web

Este proyecto tiene como objetivo principal desarrollar una aplicación web que permita calcular la nómina de los trabajadores, considerando diversos factores como el salario básico, días laborados, días de licencia, horas extras y contribuciones.

Funcionalidades Principales

  • Crear, consultar, actualizar y eliminar registros de trabajadores en una base de datos.
  • Calcular el salario devengado, deducciones y monto a pagar por cada trabajador.
  • Almacenar los datos de los trabajadores y sus liquidaciones en tablas de una base de datos PostgreSQL.
  • Validar los datos ingresados por el usuario, asegurando que cumplan con los requisitos establecidos.
  • Mostrar los resultados de los cálculos de nómina en una interfaz web amigable para el usuario.

Detalles Técnicos

El proyecto utiliza las siguientes tecnologías y herramientas:

  • Lenguaje de programación: Python
  • Framework web: Flask
  • Base de datos: PostgreSQL
  • Interfaz de usuario: HTML, CSS, Bootstrap
  • Bibliotecas y módulos: psycopg2, math

Constantes y Parámetros

El programa utiliza varias constantes y parámetros para realizar los cálculos de nómina, incluyendo:

  • Salario mínimo legal en Colombia: ${{ MINIMUM_WAGE }}
  • Valor de la Unidad de Valor Tributario (UVT): ${{ UVT }}
  • Coeficientes para calcular horas extras:
    • Horas extras diurnas: {{ EXTRA_HOUR_DAYSHIFT }}
    • Horas extras nocturnas: {{ EXTRA_HOUR_NIGHTSHIFT }}
    • Horas extras diurnas en días festivos: {{ EXTRA_HOUR_DAYSHIFT_HOLIDAYS }}
    • Horas extras nocturnas en días festivos: {{ EXTRA_HOUR_NIGHTSHIFT_HOLIDAYS }}
  • Número de días y horas promedio en un mes: {{ MONTH_DAYS }} días, {{ MONTH_HOURS }} horas
  • Porcentajes de contribuciones:
    • Seguro de salud y aportes a pensiones: {{ PERCENTAGE_HEALTH_INSURANCE * 100 }}%
    • Fondo de retiro: {{ PERCENTAGE_RETIREMENT_FUND * 100 }}%
  • Porcentajes de retención salarial según rangos de salario en UVT

Estos parámetros se utilizan en las funciones de cálculo de la nómina, asegurando que los resultados sean precisos y acordes a la legislación laboral vigente en Colombia.
